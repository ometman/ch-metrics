# Church Metrics

A comprehensive church management system for tracking and analyzing various metrics including attendance, finances, volunteers, and more.

## Features

- **Dashboard**
  - Real-time overview of key metrics
  - Visual data representation with charts
  - Quick access to important statistics

- **Metrics Input**
  - Attendance tracking
  - Financial records
  - Volunteer management
  - Educational programs
  - Digital engagement
  - Spiritual growth indicators
  - Facility utilization

- **Reports**
  - Customizable date ranges
  - Detailed breakdowns
  - Export capabilities
  - Trend analysis

- **Admin Panel**
  - Budget management
  - Target setting
  - User permissions
  - System configuration

## Technology Stack

- React 18
- TypeScript
- Tailwind CSS
- Chart.js
- Headless UI
- Vite

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/church-metrics.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Project Structure

```
src/
├── components/         # React components
│   ├── admin/         # Admin panel components
│   ├── dashboard/     # Dashboard components
│   └── metrics/       # Metrics input components
├── types/             # TypeScript type definitions
```
